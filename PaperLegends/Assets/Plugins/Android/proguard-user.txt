## File này để config thư viện không bị xóa bởi unity khi build ra mobile mỗi plugin lúc nào cũng kèm theo file proguard này nên tìm và coppy bỏ chung vào file này
## ===== Google Mobile Ads (GMA) =====
-keep class com.google.android.gms.ads.** { *; }
-keep class com.google.android.gms.internal.ads.** { *; }
-keep class com.google.unity.ads.** { *; }
-dontwarn com.google.android.gms.**
-dontwarn com.google.android.gms.internal.**

## ===== UMP (User Messaging Platform - Consent) =====
-keep class com.google.android.ump.** { *; }
-dontwarn com.google.android.ump.**

## ===== (Tuỳ chọn) Mediation adapter =====
# Nếu bạn dùng mediation, thêm keep theo tên SDK adapter bạn dùng, ví dụ:
# -keep class com.google.ads.mediation.** { *; }
# -keep class com.facebook.ads.** { *; }
# -keep class com.mopub.** { *; }
# (mỗi network có hướng dẫn keep riêng, thêm vào tương ứng)

## ===== (Tuỳ chọn) Google Play Billing (nếu có IAP) =====
-keep class com.android.billingclient.** { *; }
-dontwarn com.android.billingclient.**

## ===== (Khuyến nghị) Giảm cảnh báo AndroidX/okhttp nếu xuất hiện =====
-dontwarn androidx.**
-dontwarn okhttp3.**
